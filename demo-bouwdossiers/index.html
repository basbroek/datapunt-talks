IIIF Image Server

International Image Interoperability Framework

IIIF is geschikt voor het uniform en behapbaar (voor browsers) beschikbaar maken van grote afbeeldingen (hoge resolutie, veel MB's)

Beelden verkleinen: "…/full/800,/0/default.jpg"
Beelden draaien: "…/full/800,/90/default.jpg"
Beelden opknippen: "…/600,600,1200,1200/800,800/0/default.jpg"
